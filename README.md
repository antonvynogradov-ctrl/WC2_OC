### 4.3 Порівняння GNOME та XFCE

| Параметр | GNOME | XFCE |
| :--- | :--- | :--- |
| **Споживання RAM** | ~800 МБ — 1.2 ГБ | ~400 МБ — 600 МБ |
| **Філософія** | Сучасний мінімалізм | Класичний десктоп |

## 🏗️ Ключові висновки
Віртуалізація дозволяє безпечно тестувати різні конфігурації ОС. **GNOME** забезпечує сучасний досвід користувача, тоді як **XFCE** є оптимальним рішенням для систем з обмеженими ресурсами.
