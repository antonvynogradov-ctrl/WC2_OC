### 5. Технічне порівняння: GNOME vs XFCE
**GNOME:** Сучасний UX, підтримка Wayland, споживання RAM ~1 ГБ.
**XFCE:** Традиційний десктоп, швидка робота, споживання RAM ~500 МБ.
